# Procedural Landscapes

Creative coding project create art with procedurally generated mountain ranges using simplex noise.

## Controls

| Key        | Action           |
| ------------- |:-------------:|
| space      | Generate new landscape |
| s      | Screenshot the current landscape |  
| esc | Exit      |
